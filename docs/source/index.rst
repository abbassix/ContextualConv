.. _contextualconv:

ContextualConv
==============

Flexible, global-context conditioned convolutional layers for PyTorch. Supports:

- Per-channel **bias** (original behavior)
- Per-channel **scale**
- Full **FiLM-style** modulation (scale + bias)
